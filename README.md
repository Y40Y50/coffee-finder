# Coffee Finder

## Overview
Coffee Finder is an interactive website that helps users explore different coffee drinks and discover their favourite coffee. The site features a "Coffee-inspired" design with a smooth user experience across both desktop and mobile devices.
## User Goals
Users can search coffees, view coffee strength, and take a quiz to find a drink that suits them. The interface is designed to provide quick access to the menu while offering a fun, interactive way to learn about different blends.
## Features
Coffee finder dropdown: A responsive mobile navigation menu that toggles with a burger icon.
Coffee search: A dedicated search bar and category filters to narrow down coffee choices.
Coffee cards with images: High-quality visual cards showing coffee types like Latte, Mocha, and Espresso with descriptions.
Save favourite coffee using Local Storage: Functionality to persist user preferences across browser sessions.
Coffee quiz: An interactive multi-choice quiz that recommends a specific drink based on user taste preferences.
Dark mode: A sleek primary-color palette featuring #180202 for a premium dark-themed aesthetic.

## Technologies Used
HTML: Semantic structure for sections including Hero, Finder, Scroller, and Quiz.
CSS: Custom Flexbox layouts, media queries for mobile responsiveness, and CSS variables for branding.
JavaScript: DOM manipulation for the mobile menu toggle, icon state changes, and quiz logic.

## Deployment
The project is deployed using GitHub Pages, providing a fast and reliable live link for users to access the site.
